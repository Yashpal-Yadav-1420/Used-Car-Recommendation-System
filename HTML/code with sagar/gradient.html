    When to use Linear Gradient
  1. Soft Header
  2. Modern Hero Section
  3. Stylish Button

    When to use Radial Gradient
  1. Jab centre ko bright banana hai
  2. Jab aapko koi glow effect chahiye
  3. Jab aapko kisi card ke piche ek choti si soft light or spotlight chahiye
  4. Jab aapko chahiye ki sabki nazar center par ho
  5. Iska use aap soft and glowing look dene ke liye kar sakte ho

    Tips to remember while Radial Gradient
  1. Always use light or soft colors to bring professional look
  2. Maximum colors 2, don;t use more than that
  3. Always use colors that can blend well which gives you a modern look
  4. Don;t use pure color like dont use blue use light blue
  5. Don't use Gradient everywhere
  6. You can use gradient on a text
  7. Use Gradient on simple fonts as it gives you premium look
